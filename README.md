# Perceptron-Primal-Dual-CPP
基于 C++ 原生实现的感知机（原始形式与对偶形式）性能对比研究。

## 项目简介
本项目是机器学习感知机的简单尝试，实现了从底层向量/矩阵运算到感知机算法的完整流程。
主要对比了原始形式 $O(ND)$ 与对偶形式 $O(N^2)$ 在不同维度下的耗时差异。

## 功能特性
* 手写 Vector 和 Matrix 结构体，支持运算符重载。
* 实现了线性可分数据的随机生成逻辑。
* 支持多维空间（如 4 维、10 维）的分类测试。
* 包含毫秒级精度性能计时器。

## 实验结论
在低维海量数据（N=10000, D=2）下，原始形式远快于对偶形式。
